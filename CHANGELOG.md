# Changelog

## 1.0.0 - 2026-05-04

### Added
- 미션 등록, 수정, 삭제, 사진 인증 기능
- 미션 제한시간 직접 입력 기능
- 지도 장소 저장, 수정, 삭제 기능
- 작성자별 하트 마커 색상 구분
- 애견 동반 맛집/카페 추천 검색
- 추천 장소 네이버지도 앱 연동
- 현재 위치 공유 마커
- GPS 기반 여행 루트 기록
- 루트 전체/일자별 상세 조회
- 앨범 다중 사진 업로드
- 한국어 / 일본어 UI 전환
- 앱 아이콘 및 앱 이름 설정
- 공개 저장소용 문서 구조 추가

### Fixed
- 사진 업로드 실패 처리 개선
- 여행기록 중지 시 저장중 상태 고착 문제 수정
- 여행 루트가 0m로 저장되는 문제 완화
- 저장 마커가 추천 마커와 같은 스타일로 보이던 문제 수정
- 지도 안내 패널이 마커 정보를 가리는 문제 개선
- 하단 시스템 내비게이션 버튼에 UI가 가리는 문제 개선

### Security
- Google Maps, Firebase, Cloudinary 설정을 하드코딩하지 않고 빌드 설정으로 분리
- Firebase 설정 파일과 빌드 산출물을 `.gitignore`에 추가
