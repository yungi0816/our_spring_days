# 테스트 계획서

## 테스트 대상

- 미션 생성 / 수정 / 인증
- 지도 장소 저장 / 수정 / 삭제
- 추천 장소 검색 / 네이버지도 연결
- 여행기록 시작 / 중지 / 루트 상세
- 앨범 다중 업로드
- 사용자 전환
- 하단 안전영역

## 자동 테스트

| TC ID | 기능 | 테스트 내용 | 기대 결과 | 결과 |
|---|---|---|---|---|
| TC-001 | 사용자 전환 | 저장된 사용자 복원 | 마지막 선택 사용자 표시 | PASS |
| TC-002 | 사용자 전환 | 사용자 변경 저장 | SharedPreferences에 저장 | PASS |
| TC-003 | 미션 Provider | 진행 중 미션 여러 개 추가 | 목록에 모두 유지 | PASS |
| TC-004 | 앱 로드 | 앱 스모크 테스트 | 첫 화면 렌더링 | PASS |

## 수동 테스트

| TC ID | 기능 | 테스트 내용 | 기대 결과 |
|---|---|---|---|
| MT-001 | 여행기록 | 시작 후 100m 이상 이동하고 중지 | 루트에 2개 이상 포인트와 거리 표시 |
| MT-002 | 여행기록 | 시작 직후 중지 | 시작/종료 지점이 저장되고 저장중 고착 없음 |
| MT-003 | 지도 | 지도 길게 눌러 장소 저장 | 하트 마커 표시 |
| MT-004 | 지도 | 저장된 마커 수정 | 변경된 이름/주소/카테고리 반영 |
| MT-005 | 지도 | 저장된 마커 삭제 | 지도에서 마커 제거 |
| MT-006 | 지도 | 추천 장소 마커 터치 | 네이버지도 앱 또는 웹 열림 |
| MT-007 | 앨범 | 사진 여러 장 선택 후 저장 | 사진별 앨범 항목 생성 |
| MT-008 | 하단 UI | Galaxy 내비게이션 버튼 환경 확인 | 하단 탭이 가려지지 않음 |

## 테스트 명령어

```bash
flutter analyze
flutter test
```

## 실기기 테스트가 필요한 이유

GPS, 백그라운드 위치, 알림, 갤러리 저장, 네이버지도 앱 실행은 에뮬레이터보다 실제 Android 기기에서 확인하는 편이 정확합니다.
